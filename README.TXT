Saints Row IV Music Extractor
Extracts the music tracks from the Saints Row IV radio stations.

- DEPENDENCIES -
Minimaul's Saints Row IV Tools - http://www.saintsrowmods.com/forum/threads/minimauls-saints-row-iv-tools.3188/
Toad King's BNK_PC Extractor - http://www.saintsrowmods.com/forum/threads/bnk_pc_extractor-ripping-audio-files.55/
WW2OGG - http://www.hcs64.com/vgm_ripping.html
REVORB - http://yirkha.fud.cz/progs/foobar2000/revorb.exe

- HOW TO USE -
1. Download all the dependencies. (above)

2. Extract/copy the dependencies to the tools folder.
	Example Folder Structure: ( [ and ] indicates a folder )
		- [Saints Row IV Music Extractor]
			- [tools]
				- [bnk_tools]
				- [ThomasJepp.SaintsRow-rev22]
				- [ww2ogg019]
				- revorb.exe
Remember, this is just an example, the script is very flexible. Just throw them in there somewhere (extracted) and it should work.

3. run sr4me.cmd
